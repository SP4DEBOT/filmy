# 🎬 Filmy

A modern movie discovery application built with **React**, **Vite**, and the **TMDB API**. Browse trending movies, save your favorites to a watchlist, and organize them with filtering and sorting features.

---

## 📌 Features

- 🎥 Browse trending movies
- ❤️ Add and remove movies from your watchlist
- 🔍 Search movies in your watchlist
- 🎭 Filter movies by genre
- ⭐ Sort movies by rating
- 💾 Watchlist stored using Local Storage
- 📱 Responsive design

---

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Axios
- TMDB API

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/SP4DEBOT/filmy.git
```

### Navigate to the project

```bash
cd filmy
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open **http://localhost:5173** in your browser.
